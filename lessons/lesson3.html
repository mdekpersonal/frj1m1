🧑‍🤝‍🧑 Leçon 3 : Mes amis et objets de la classe ✏️

🖊️ 🎒 📒 🧑‍🤝‍🧑 Voici quelques objets de la classe : un stylo, un cartable, un cahier, un ami, une amie.

Répète après moi : 🖊️ un stylo, 🎒 un cartable...

🔎 Où est ton stylo ?

Dis « Voici mon stylo. Il est bleu. »

🧸 Activité BYOO

🧸 🖊️ 🎒 📒 Montre un objet de ta classe ou un jouet.

Dis : « Voici mon stylo. Il est bleu. » (ou adapte avec ton objet)

  • 🎒 Voici mon cartable. Il est rouge.

🗣️ Exercice oral

❓ Tu demandes : Où est la gomme ?

💬 Réponds : « La gomme est sur la table. » (ou autre réponse simple)

  • ❓ Tu demandes : Où est le cartable ?
    💬 Réponds : « Le cartable est sous la chaise. »
  • ❓ Tu demandes : Où est le stylo ?
    💬 Réponds : « Le stylo est dans la trousse. »
  • ❓ Tu demandes : Où est le cahier ?
    💬 Réponds : « Le cahier est sur le bureau. »
  • ❓ Tu demandes : Où est le nounours ?
    💬 Réponds : « Le nounours est dans le cartable. »
  • ❓ Tu demandes : Où sont les livres ?
    💬 Réponds : « Les livres sont dans la bibliothèque. »

📖 Lecture

📚 Lis :

  • 🖊️ Voici mon stylo.
  • 🔵 Il est bleu.

✍️ Écriture

Écris les phrases dans to cahier.

Exemples :
  • 🖊️ Voici mon stylo. Il est bleu.
  • 📒 J’aime mon cahier. Il est vert.
  • 🧽 Ma gomme est rose.

🪞 Monologue devant le miroir

🪞 Écoute ce petit monologue, puis répète-le devant un miroir. Répète plusieurs fois jusqu’à pouvoir le dire tout seul, sans l’aide de l’audio !

🧠 Quiz

1. 🔵 Quel objet est bleu ?

Le stylo
La gomme
Réponse : Le stylo.
Explication : Dans l’exemple, le stylo est bleu.

2. ✏️ Quel mot correspond à un objet pour écrire ?

Stylo
Gomme
Réponse : Stylo.
Explication : On écrit avec un stylo.

3. 🧽 À quoi sert une gomme ?

À écrire
À effacer
Réponse : À effacer.
Explication : On utilise une gomme pour effacer.

4. 🧑‍🤝‍🧑 Complète la phrase : Voici mon ____.

ami
table
Réponse : ami.
Explication : On présente souvent un ami ou un objet avec « Voici mon ... ».

5. 🔵 Laquelle de ces phrases est correcte ?

Il est bleu.
Il est bleue.
Réponse : Il est bleu.
Explication : « Bleu » est la forme correcte pour le masculin singulier.
