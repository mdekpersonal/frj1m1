🧑‍🎓 Leçon 2 : Parler de soi avec les pronoms 🗨️

👦 Aujourd’hui, nous allons apprendre les pronoms personnels : 🧑 Je, 🫵 tu, 👦 il, 👧 elle, 🙋‍♂️ moi, 👉 toi.

Répète après moi : 🧑 Je, 🫵 tu, 👦 il, 👧 elle, 🙋‍♂️ moi, 👉 toi.

💬 Maintenant, dis « J’aime… » avec un objet ou un animal.

Par exemple : 🐱 « J’aime le chat. »

🗣️ Essaie maintenant !

🧸 Activité BYOO

🧸 Choisis un objet ou une image d’animal.

🗣️ Dis : « J’aime … » le nom de ton objet ou animal.

🔁 Répète plusieurs fois.

🗣️ Exercice oral

👂 Ecoute bien ces questions :

  • 🍫 Tu aimes le chocolat ?
  • 👍 Oui, j’aime le chocolat.
  • 👎 Non, je n’aime pas le chocolat.

🔁 Répète chaque phrase.

📖 Lecture

📚 Lis ces phrases :

  • 🐱 J’aime le chat.
  • 🍫 Tu aimes le chocolat ?

✍️ Écriture

📝 Complète :

J'aime .... / Je n’aime pas ...

Écris deux phrases sur tes goûts.

Exemples :
  • 🐱 J'aime les chats.
  • 🥬 Je n’aime pas les épinards.
  • 🍫 J’aime le chocolat.
  • 🐍 Je n’aime pas les serpents.

🪞 Monologue devant le miroir

🪞 Écoute ce petit monologue, puis répète-le devant un miroir. Répète plusieurs fois jusqu’à pouvoir le dire tout seul, sans l’aide de l’audio !

🧑 Je m’appelle Ali.

🧠 Quiz

1. 🙋‍♂️ Quel pronom correspond à « moi » ?

Je
Tu
Réponse : Je.
Explication : « Moi » correspond au pronom « Je » pour parler de soi.

2. 🐱 Choisis la bonne phrase :

J’aime le chat
Tu aime la chat
Réponse : J’aime le chat.
Explication : On dit « J’aime le chat » car le verbe s’accorde avec « je » et « le chat » est masculin.

3. 🍫 Complète la phrase : « Je ___ le chocolat. »

aime
aimes
Réponse : aime.
Explication : Avec « je », on utilise « aime » (je aime → j’aime).

4. 🫵 Quel pronom utilise-t-on pour parler à un ami ?

Tu
Il
Réponse : Tu.
Explication : On utilise « tu » pour s’adresser directement à quelqu’un.

5. 👦 Laquelle de ces phrases est correcte ?

Il aime les chats
Il aimes les chats
Réponse : Il aime les chats.
Explication : On écrit « Il aime » (sans s) car le verbe s’accorde avec « il ».
